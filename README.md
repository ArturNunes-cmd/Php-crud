# 📄 Sistema de Gerenciamento de Pizzas (CRUD em PHP)

Este projeto é um sistema simples desenvolvido em PHP puro, que permite cadastrar, listar, editar e excluir pizzas.
Os dados são armazenados em um banco de dados MySQL, utilizando operações CRUD (Create, Read, Update, Delete).
O sistema funciona em ambiente local (ex: XAMPP, WAMP ou Laragon) e realiza conexão com banco de dados SQL.

## 🚀 Funcionalidades

### ✔️ 1. Cadastrar Pizza
Permite registrar uma nova pizza contendo:
- Nome da pizza
- Sabor
- Tamanho
- Preço
Os dados são enviados para o banco de dados MySQL.

### ✔️ 2. Listar Pizzas
Exibe todas as pizzas cadastradas diretamente do banco de dados.

### ✔️ 3. Editar Pizza
Permite atualizar as informações de uma pizza já cadastrada.

### ✔️ 4. Excluir Pizza
Remove uma pizza do banco de dados.

### ✔️ 5. Integração com Banco de Dados
- Conexão com MySQL
- Uso de comandos SQL (INSERT, SELECT, UPDATE, DELETE)
- Manipulação de dados via PHP

## 🛠️ Tecnologias Utilizadas
- PHP
- MySQL
- HTML
